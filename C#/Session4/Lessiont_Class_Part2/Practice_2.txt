Menu:

1.Nhập thông tin học sinh.
2.Nhập điểm của học sinh.
2.Hiển thị thông tin học sinh.
3.Tính điểm trung bình môn của học sinh đó.
4.Xếp loại học lực của học sinh.
5.Thay đổi thông tin học sinh.
6.Thoát

Yêu cầu: 
	1. Student(hoc sinh): 
		public string Name
		public int Age
		public string Address
		public string Email
		public Mark[] Marks
	2.Mảng Marks của Student khởi tạo là 1 mảng class Mark với 2 trường:
		public string NameSubject
		public int Mark (nhỏ nhất = 0 và lớn nhất là 20)
		Khởi tạo với 5 subject sau: Toán, Văn, Lý,Hóa,Sinh
	3.Xếp loại học lực như sau:
		[0,8] Yếu
		(8,12] TB
		(12,16] Khá
		(16,20] Giỏi
